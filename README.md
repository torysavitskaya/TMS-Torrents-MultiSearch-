**Torrents MultiSearch (Unofficial MV3 Port)**

Неофициальная адаптация расширения Torrents MultiSearch под стандарт **Manifest V3**.

Оригинальный проект был заброшен автором на версии 3.1.2 и перестал работать в современных браузерах из-за отказа от Manifest V2. В этой версии код обновлен и адаптирован для соответствия новым требованиям безопасности и архитектуры Chrome.

**Что нового:**
*   Полная миграция на Manifest V3.
*   База — оригинальная версия 3.1.2.
*   Восстановлена работоспособность в актуальных версиях Chrome/Edge/Brave.
